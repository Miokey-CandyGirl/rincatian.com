个人中心 退出登录

🔧 社区个人中心关闭按钮修复测试

🐛 问题描述

原问题：

只有社区页面点击"个人中心"对话框后，右上角的叉号无法点击进行关闭

问题原因：

• createNewModal 中的关闭按钮调用 closeModal()（无参数）

• community.js 重新定义了 closeModal(modalId)（需要参数）

• 函数冲突导致关闭失败

修复方案：

• 创建专门的 closeNewModal() 函数

• 修改关闭按钮调用新函数

• 添加事件监听器而非内联 onclick

📋 测试步骤

请按以下步骤进行测试：

1. 点击下方"快速登录"按钮
2. 登录后点击导航栏右上角的"个人中心"
3. 在弹出的个人中心对话框中，点击右上角的 ✕ 按钮
4. 验证对话框是否能够正常关闭
5. 重复测试几次确保稳定性
