🔧 用户信息显示样式修复测试

🛠️ 修复说明

问题：

社区页面的用户信息显示为换行形式（垂直布局），与其他页面的单行显示（水平布局）不一致

根本原因：

community.css 中的 .user-info 样式覆盖了 styles.css 中的默认样式：

// 问题样式（已修复）
.user-info {
    display: flex;
    flex-direction: column; // 导致垂直布局
}

// 修复后样式
.user-info {
    display: flex;
    align-items: center; // 水平居中对齐
    gap: 0.8rem;        // 保持与其他页面一致的间距
}
