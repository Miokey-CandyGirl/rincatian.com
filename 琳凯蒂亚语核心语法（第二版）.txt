【规则2：词类】
2.1 按数量：有限词（代词、数词、连词、助词、抒情词）；无限词（名词、动词、形容词、副词、声叹词）
2.2 按来源：外来词（需要将外来语的拉丁拼写方案（例如日语的罗马字，汉语拼音）在形态上转换成符合田语的41个字母的形态，例如英语“chat”转写成“ĉat”）；原有词（类似于成语似的语言压缩包，例如，gaŋ-d-yodene：说的也是）；标准词（田语中规定且固定的3000个名、动、形、副词根+所有有限词，声叹词不属于标准词）
2.3 抒情词：包含强调词（如ce），领语中的牵词，结语中的止词。所有抒情词均可做独立成分。

【规则3：标准词（名、动、形、副）转义词尾规则】
3.1 标准词中的名词、动词、形容词、副词只拥有唯一一种原生词性，若需转为其他词性（转为名、动、形、副其一）需要添加相应的转义词尾。原生词性无需添加转义词尾（如果添加也不为错只是多余）。
例词：mono（原生名词“姓名”，无需写成monoo）转义为动词写作monoe（命名，非mone）。
3.2 田语的名词复数表示的是集体属性，而非类似于英语，比如“soseon”，正确理解为“学生们”，并非在翻译“两个学生”时，写成“nim soseon”，正确应写为“nim soseo”。“sose”是动词“学习”。再例如，“inon”为“人群，大众”，“in”为“人、人类”。
3.3 专有名词不属于标准词，由于它们本身就是名词，所以无需名词转义词尾。
3.4 外来词需要添加相应的转义词尾用以提示词性。
3.5 转义词尾表：
名词：-o（集体属性复数：-on）	动词：-e、-le等16种体态词尾
形容词：-i（弱形容词：弱情感、含蓄；状态形容词）
-a（强形容词：强烈、夸张、鲜明；性质形容词）	副词：-u（通常修饰动词或形容词）
-li（通常修饰其他成分或句子）
口语中，强弱形容词以及副词的两种词尾作用的分界并不明显，主要由说话者依情感自己决定使用哪种。

【规则4：方位词规则——静态】
4.1 方位词做为名词方位词尾可与-o、-on词尾叠加。例如deko-n-qin→dekonqin（一些桌子前）。
4.2 名词方位词尾之构词规则：dekowo为“桌子上（不考虑接壤与内部关系）”，dekowāwo为“悬浮于桌子上方”，dekoθinē为“桌子内部接壤处”。
例句：	dekowo’lio ez libo. 桌子上有书。libo’lio ez dekowo. 书在桌子上。
4.3 方位词能直接做名词使用。例如，Elasï’l ez tāyaŋ. 东边有太阳。
4.4 动词与方位词可构建形成方动词组（方动词组=方位词+中心动词，中间留有空格，动词体态词尾放置于动词上）充当述语。例如，“hô mide”为“照看”，“hô”本身含义为“在……后面”，“mide”为“见”。

【规则5：趋向词规则——动态】
5.1趋向词本身可充当述语成分作动词（可直连宾语）表示主语的位移状态。例如：计划位移：toge（计划前往）；位移中：toze（正在路上）；位移完成：tole（已经到达）。
例句：Win toge soselio. 我们打算去学校。（toge是to的计划式）
5.2趋向词也可做为谓语动词的词尾，构成新的动趋复合动词来充当述语（复合动词=谓语动词+趋向词，中间不留空格，动词体态词尾放置于复合动词末尾）表示动作的方向等。
例词：	midewotoze 正在看上去（mide：见；woto：上去）
pêlētoze 去玩了（侧重在去玩的路上，即开始执行中，ze为动词进行式） pêlētole 玩去了（侧重事后解释，le为动词完成式） pêlēge 打算玩（ge为动词计划式）
5.3 名词也能接趋向词。①表示起始地点或时间的在趋向词末尾加“r”尾音或者在名词（包含含有方位词的名词）后以及含有尾音的趋向词后加“r”轻音（通常译为“从……、自……”），如果是短语或句子，使用“’ri”助词；②表示目标地点或时间的在趋向词或名词（包含含有方位词的名词）末尾加“re”（通常译为“到……、当……”），如果是短语或句子，使用“’re”助词。例如“从这件事情说起”为“ko mono’ri”。
例词：①soselioĉulār homonēre从学校出来，到家里去；②togenr qigenre从昨天到今天”。

【规则6：性状动词规则】
用来充当述语成分，包含两种，一是表示性质或判断的“es”（类似于汉语“是”），二是表示状态或存在的“ez /eθ/”（表示在/有）。性状动词不可与其它动词或形容词构成复合述语（即性状动词只可单独做述语成分），但可被副词修饰。性状动词的具体逻辑如下：
6.1性质动词“es”——等同性（表示主语的性质或判断主语是什么的连接桥梁，后接宾语）
①等同的：A es B，A＝B，意为“A是、等于、等同于B”；
②近似的：A ese B，A≈B，意为“A像、可能是、近似于B”；【像不像为esene】
③不同的：A nes B，A≠B，意为“A不是、不等于、不同于B”；
④不定的：A esne B，A?B，意为“A是否为、是不是B”；
⑤所有的：A caŋes B，A≌B，意为“A总是、全部是、都是B”；
⑥肯定的：A ces B，A==B，意为“A一定是、肯定是、绝对是；就是B”。【否定为cenes】
6.2状态动词“ez”——现实性（表示主语状态或者存在性的连接桥梁，后接宾语）
①现实的：A ez B，A∈B，意为“A在，置于，存在于B”；B∈A，意为“B有，拥有A”；
②可能的：A eze B，（A∈B）∪（A∉B），意为“A可能于B”；（B∈A）∪（B∉A），意为“B可能有A”；
③不可能的：A nez B，A∉B，意为“A不于B”；B∉A，意为“B没有A”；
④不定的：A ezne B，A?B，意为“A是否在、在不在B”；B?A，意为“B是否有、有没有A”；
⑤所有的：A caŋez B，A≌B，意为“A总在B”；B≌A，意为“B总有、全部有，都有A”。【“总”表示时间上的任何时候】
⑥肯定的：A cez B，A==B，意为“A肯定在；就在B”；B==A，意为“B一定有、绝对有；就有A”。【否定为cenez】

【规则7：形容词的比较规则】
7.1 形容词词尾分为原级、相同级、比较级、最高级4种。
原级（做修饰词）	相同级（与……一样）	比较级（比……更……）	最高级（最……）
-i或-a	-yixoli（简写为：-xol）	-tē	-sā
注：除原级外的三个词尾已具有形容词属性，所以无需形容词转义词尾“-a”或“-i”。
7.2 含有相同级、比较级、最高级的形容词做述语时，形容词的体态词尾放在后面。
例词：belasālie 曾经最美（lie形容词做述语时不做修饰词的完成式）
7.3 当处理比较句（含有相同级、比较级、最高级形容词的句子）时，宾语使用直接宾语助词“-ō”。
例句：Wi belatē ŝi’ō. 我比她漂亮。（bela是原生词，没有添加转义词尾-a，故无需写成beltē；省略了形容词的一般主动式ye）。

【规则8：声叹词规则】
声叹词（拟声词、拟态词、叹词）统一词尾为“-h”，做独立成分。
例如“叮咚”为“diŋdoŋh”，闪耀的样子为“kilāh”，“啊”为“ah”。

【规则11：连词规则】
连词的作用是用逻辑连接词（连词）连接两个及以上的句子或者词语。
11.1常用的连词
①联合关系	hâ（和、与、且）；kowe（此外）；giza（总之）；ole（并且、而且）
②对立关系	kona（或者，还是）；sedo（但是）；nes……cenes（既不……也不……）；vakan（尽管）；nega（相反）
③因果关系	qavi（因为）；gosi（所以）；silo（因此，那么）；kuni…silo…（如此……以至于……）
④条件关系	rugo（假设）
⑤目的关系	poroŋ（为了）

【规则12：助词规则】
句子成分有专门的提示助词做标记，使语序完全可以凭借不同母语者的表达习惯或者强调某个成分将其提前（提前的部分作为领语），而不会影响句意表达。
12.1主语靠“'ô（施语为可自发动作，wô的缩写）、'j（施语为不可自发动作，jio的缩写）、'l（事物性质、状态、抽象概念做施语时，lio的缩写）”主语助词提示。当一个句子充当主语时改用“'fo”；
12.2谓语靠动词词尾或形容词词尾或补语助词做谓语成分的标记；
12.3当宾语有直接、间接区分时，需要标记间接宾语，间接宾语助词为“'ô”，直接宾语助词省略。
直接宾语提示助词“'ō”通常用来处理处置式（比如汉语把字句的翻译）、使动语态、被动语态、迫动语态、兼语句的表达，其他情况通常省略。当一个句子充当宾语时改为“fō”。
12.4补语的提示助词为“'r”，当一个句子充当补语时，改为“ve”。
12.5定语提示助词为“'a”，当一个句子充当定语时改为“'da”。时间词或者表示时间的短语、句子作定语用助词“'xa”，地点为“'ka”。
12.6状语提示助词为“'u”，当一个句子充当定语时改为“'du”。时间词或者表示时间的短语、句子作状语用助词“'xu”，地点为“'ku”。
12.7方式状语用助词“'peyoŋ”，同时省略状语提示助词“’u”或“’du”。
12.8频率状语（频率是指同一动作在一定范围或时间内出现的数量的多少）用助词“'gu”或者“'gun（表示多次）”。例如“1天两次”为“1 tin 2'gu”;“1天多次”为“1 tin'gun”。同时省略状语提示助词“’u”或“’du”。
12.9目的状语“'poroŋ”（助词）：动作实施计划实现的目的、前往的目标人或物。计划实现的某种状态。译为“为、为了、为了……目标而努力”。“'poni”（助词）：动作实施已经实现的目的、到达的目标人或物。已经实现的某种状态。相近词“pone（动词）”译为“给、于、给予”。

【规则14：谓语动词和形容词的变化规则】
根据使之变化的对象的不同，变化分为自变和他变，在不知道是否为自变的情况下均以自变考量。自变是指不受外界影响，自发产生的变化，一般只对主语负责，无需宾语；他变是指受到外界的影响而产生变化，一般需要有宾语。
再者，根据变化时是否产生位移可分为静变（静态转变）和动变（动态转变）。静变是指人或物的性质或状态发生变化的结果。也叫自我趋向。静变一般变化时所需的时间较长，肉眼难以观察其变化过程,侧重变化的结果；动变是指人或物的性质或状态发生变化的过程。动变一般变化时所需的时间较短，在短时间内可以看见其变化过程，侧重变化的过程。
前一组和后一组可两两组合，以突出更加丰富的变化含义。形容词表示变化的表述如下
自静变（主动完成式）：lie；自动变（主动进行式）：zie；
他静变（被动完成式）：liē；他动变（被动进行式）：ziē。

【规则15：句式规则】
任何形式的句子都以标准句式为模板，如果有助词，则可以依需求调整成分顺序。
15.1标准句 = 领语 + 主语（定语+施语）+ 谓语（状语+述语+补语）+ 宾语（定语+受语）+ 结语。
也可理解为：= 领语 + 定语 + 施语 + 状语 + 述语 + 补语 + 定语 + 受语 + 结语。
15.2领语 = 牵词（仅一个单词）/ 被提前的某个成分（/表示或者，下同）
15.3主语 = 定语 + 施语 / 短语 / 句子 + 主语助词
15.4定语 = 体词（名词、代词、数词）/ 形容词 / 形容词性短语 / 形容词性句子 + 定语助词
15.5施语 / 受语 = 体词 / 体词短语（例如同位语短语）
15.6谓语 = 状语 + 述语 + 补语 + 补语助词（述语包含动词或形容词词尾）
15.7多重状语顺序：时间状语 + 空间状语 + 方式状语 + 频率状语 + 心理动词 ++ 述语（动词/形容词）
15.8补语 = 数词（表示动作实施的次数） / 形容词 / 名词 / 句子
15.9直接宾语 / 间接宾语 = 定语 + 受语 / 短语 / 句子 + 宾语助词
15.10结语 = 前止词 + 后止词
15.11 任何一个标准句中，不允许出现两个及以上的谓语（复合动词算做一个），除非用连词连接或者作为标准句中的状语、定语、主语等其他成分出现。

【规则16：领语规则】
16.1祈使句通过领语完成。例如牵词celoŋ（请）、nuden（禁止）、den（允许）、duhaden（不许，不要）、haden（建议）、nen（难道）。
16.2强调句中强调某个词时，用强调词ce，置于被强调词前做前缀使（cebela：非常美）。ce也可以置于句首做领语，强调整句话或者主语。
16.3强调某一语法成分时，也可把整个成分连带它的助词一并提前做领语。领语与后面的成分需要用逗号隔开。如果提前的成分中只含有一个单词，其助词可以省略（不强制）。
16.4指示代词的强调形式：“θi”为“这，那”；“θin”为“这些，那些”，它们都做领语。
16.5“eca”用来强调句中的形容词做领语。
16.6“hoseela”用来强调名词或形容词，无需做领语，加在被强调名词或形容词前，表示“多么”，例如“多么美好的一天啊！”。“hoseeli”用来强调动词或副词，无需做领语，加在被强调动词或副词前，表示“多么”，例如“他多么想成为一名科学家啊！”。

【规则17：谓语多动词规则】
17.1 连动句
17.1.1表目的：小明去图书馆看书。方案：“去图书馆”用目的状语助词标记修饰动词“看”。
17.1.2表顺序：小红开门走出去了。方案：用“hâ（和）”连接“开门”和“走出去”两个动作。
17.1.3表方式：小张坐飞机去出差。方案：“坐飞机”用方式状语助词标记修饰动词“去”。
17.2 同一施动者的共动句
17.2.1表示同一施动者同时进行两个及以上的动作时，用“yixo”这个连词连接这些不同的动作（连接动词或动宾短语），译为“同时、一起、共同、一边…一边、并且”。
17.2.2表示同一施动者在进行一个动作的过程中临时做了另一个动作时，用“时间点的词/短语/句子+xu+谓词”，例如“小明在吃饭的时候电话响了。”翻译为“Elovoto’jio Xiomiŋ’ô tabeze’xu haŋdole”。
17.3 不同施动者的共动句
17.3.1【着重动作，执行的动作是相同的，但人不一定在一起】表示不同施动者同时进行一个相同的动作时，用“yixoli”这个副词表示，放置于执行同一动作的动词前作状语修饰动词，施动者之间用“hâ”连接，译为“与……一起、一同”。
17.3.2【着重人，表示两人在一起】用“yixo”连词直接连接施动者即可。

【规则18：结语规则】
18.1陈述句（yo结尾）、疑问句（ne结尾）、感叹句（ga结尾）以及丰富的情感表达（包括类似于日语的尊敬语，只是不用动词表达）通过结语完成。
18.2结语结构为“前止词+后止词”，前止词反应句子的情感，后止词反应句子的类型（如ga、yo、ne）。【口】日常中只使用ga、yo、ne即可，除非在特殊的场合下（面试、商务、演讲等）或者需要准确表达情感时需要用指定的结语（因为结语表示的情感非常丰富和细化，复杂度远超日语的尊敬语，面对不同场合、不同人物、不同语境等需要选用最适合的结语，结语类型统计有上百个之多，所以学习者可以忽略掉复杂的结语）。
18.3如下是基本结语（有的前后止词融合了，有的省略了后止词）：
陈述句的基本结语（前止词+后止词）
①表新动作的出现：teyo	②表确认本来如此：qoyo
③显而易见的动作：meyo	④表示略带夸张的语气：kayo
⑤表示已经到达某种状态：leyo	⑥用于略有强调意味的句子中：handio
⑦使语言变得委婉可爱：mio	⑧用于双重否定句中：gio
一般疑问句的基本结语（前止词+后止词）
①不带止词，加上问号，语调上升，用以表示疑问效果。	②一般是非问（吗），在陈述语句后直接加ne。
③表示半信半疑的语气（吧）：ane	④表示委婉语气（啦）：nie
⑤使语言变得可爱：ya	⑥用以想得到回答者的回复时：nānna
⑦表示着重语气：qiaŋgu	
特殊疑问句的基本结语（前止词+后止词）
①问什么：ne	②问性质或状态：hananne
③问原因：yen	
选择疑问句的基本结语（前止词+后止词）
①一般的选择疑问：nē（选择某个动作时） / no（选择某个事物时）	②委婉的选择疑问，有的表示询问他人意见：kajinē
反义疑问句的基本结语（前止词+后止词）
①在反义疑问句中，用une。	
感叹句的基本结语（前止词+后止词）
①表示喜悦：handêga	②表示讥笑和鄙视：nonka
③表示痛楚和悲哀：heka	④表示诧异：nēnga
⑤表示叹息和感慨：nēnka	⑥表示醒悟：soga
⑦表示愤怒和厌恶：nonga	⑧表示恐惧：oŋga
⑨表示惊讶：cesioŋga	
18.4反义疑问句的答句：表肯定的结语为“esdē”，表否定的结语为“nesdē”。

18.5 表示情态、推测、义务、能力、愿望等的能愿前止词
能愿前止词通常用于表达语气较为委婉的陈述句或疑问句中。分为“未然态”、“必然态”、“命令态”、“意愿态”、“估价态”、“能力态”6种。部分能愿前止词可做心理动词且可单独做述语。例如，deθi（想，想念）。能愿前止词的否定形式只需要加上否定前缀“ne-”即可，命令态无否定形式。
例句：	Wi pêlē kompio neghiyo. 我不会玩电脑。(我 玩 电脑 不会. neghiyo=前止词gâhi的否定negâhi+后止词yo）
Wi deθize ni. 我想你了。(deθize表示正在想念）
18.5.1未然态：表示动作暂未发生，但在未来具有发生的可能性。
①sēē：表示未来一定发生，但目前未发生。【要（推测、将来）、将（将要、即将）、可以、得以、可、能；会（将来）】
②sēro：未来可能会发生，也可能不发生。【可能、会（可能性）】
18.5.2必然态：强调动作发生的必然性。
①devi：情理上必然或必须如此。【应、应当、按理说、该、应该、需要、犯得着、有必要】
②deven：现在或未来的动作必然发生。【必须、肯定、一定】
18.5.3命令态：命令他人实施行动。
①正向命令：handen【最好、要（要求、命令）】
②强制正向命令：ceden【必须、得、必然】
③反向命令：duhanden【最好别、不要（要求、命令）】
④强制反向命令：nuden【禁止】
18.5.4意愿态：表示主观意愿：想要达到某种状态或者实现某种动作。
①意要行为：deθi【要（表示意愿、请求、习惯）；想、想要、希求、要想】
②愿望行为：voli【愿意；愿（祝愿、想要、可能性、许可）、期盼、盼望】
③需求行为：manniō【需要】
④强烈欲望行为：dāer【敢、肯】
18.5.5估价态：表示动作实现之后的价值程度。
①可以获得好的结果：mezi【值得】
②方便获得好的结果：poro【便于、易于】
18.5.6能力态：表示主观拥有的能力。
①ghi【会（表能力）】
②povi【能、可以】

【规则19：学术研究的占位规则】
名词原形：mon；动词原形：em。主语原形：mōn；谓语原形：doze；宾语原形：mōm。名词也可用通用代词“xi”占位。
原形的作用是当句子中没有相应语法成分时用来占位。只在学术研究中使用。

【规则20：词尾和助词书写规则】
20.1除特殊规定外，一个词语禁止使用多重词尾或助词。
20.2词尾直接附着在单词的后面，“-”必须省略。
20.3助词有两种写法，第一，助词接在相应的成分后面，“'”保留，与前面的单词不留有空格。第二，“'”删除，与前面的单词留有空格。例如，“libo’j”或者“libo j”。
20.4在不影响句意的表达下，语言使用者可省略任何助词和任何成分（靠主谓宾等语序判断成分），以减轻规则，避免影响日常交流，同时保障语言的准确性和少歧义性。
20.5在日常使用中，数词和代词后可省略助词。
